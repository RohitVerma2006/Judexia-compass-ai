

# Judexia – "Train Your Legal Mind with Intelligent Systems"

An AI-powered LegalTech learning platform — hackathon prototype, fully frontend-simulated.

---

## Phase 1: Foundation & Design System
- Set up midnight blue gradient theme with glassmorphism cards, gold accents, electric blue highlights
- Embed the uploaded Judexia logo on login/signup and as sidebar branding
- Use reference images (scales, library) as hero/background imagery on landing page
- Serif headings + sans-serif body, smooth animations throughout

## Phase 2: Authentication (Simulated)
- Signup page (Full Name, Email, Password, Confirm Password) → stores in React state
- Login page → accepts any credentials, redirects to dashboard
- Logout clears state, disclaimer label about production security

## Phase 3: Dashboard & XP System
- Sidebar with 9 modules (AI Case Studio, AI Legal Mentor, Quiz Mode, Case Library, Document Simplifier, Community Forum, Lawyer Consultation, Legal Roadmap, Draft Notice Generator)
- Top bar: user name, XP counter, level badge (Law Rookie → Advocate → Senior Counsel), logout
- XP persists in state and updates across all modules

## Phase 4: AI Case Studio
- Split layout: topic input + scrollable chat window
- Keyword-detected structured responses (case title, scenario, evidence, articles, landmark references)
- Follow-up analysis prompts with evaluation and XP awards

## Phase 5: AI Legal Mentor
- Dedicated chatbot interface for legal Q&A
- Structured responses (definition, explanation, example, case reference, recap)
- Keyword matching for common legal topics, awards XP

## Phase 6: Document Simplifier
- Drag & drop upload UI with 2-second animated processing
- Displays summary, key points, risk clauses, suggested action
- Labeled "Prototype Simulation", awards XP

## Phase 7: Community Forum
- Thread list with upvotes (live increment), comments, Create Post modal
- Pre-populated sample legal threads

## Phase 8: Quiz Mode
- 5 MCQs, 60-second timer, score display, correct answers, XP
- Leaderboard with mock users + current user ranking

## Phase 9: Remaining Modules
- **Lawyer Consultation**: Category + time slot booking with confirmation modal
- **Legal Roadmap**: Goal selection → structured weekly plan generation
- **Draft Notice Generator**: Type selection → formatted notice preview with simulated download

## Phase 10: Polish
- Multi-language toggle (English/Hindi partial simulation)
- Footer disclaimer on all pages
- Responsive design and animation polish
